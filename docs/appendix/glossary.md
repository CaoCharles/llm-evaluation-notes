# 術語表

LLM 評測領域常用術語對照與解釋。

## A

**Accuracy（準確率）**
: 預測正確的樣本數佔總樣本數的比例。

**Adversarial Examples（對抗性樣本）**
: 刻意設計用來測試模型穩健性的輸入樣本。

**Answer Relevancy（答案相關性）**
: 評估回答與問題相關程度的指標。

## B

**Bias（偏見）**
: 模型輸出中對特定群體的不公平傾向。

**Benchmark（基準測試）**
: 用於評估模型表現的標準化測試集合。

## C-D

**Context（上下文）**
: 提供給 LLM 的背景資訊，用於生成回答。

**Context Precision（上下文精確度）**
: 檢索結果中相關資訊排序是否靠前。

**CoT (Chain of Thought)（思維鏈）**
: 引導模型逐步推理的 Prompt 技術。

## E-F

**Embedding（向量嵌入）**
: 將文字轉換為數值向量的表示方法。

**Exact Match（完全匹配）**
: 模型輸出與標準答案完全一致的比例。

**F1 Score**
: Precision 和 Recall 的調和平均數。

**Faithfulness（忠實度）**
: 回答是否忠於提供的上下文資訊。

**Few-shot**
: 在 Prompt 中提供少量示例引導模型。

## G-L

**Groundedness（可追溯性）**
: 回答中的陳述是否可追溯到來源資訊。

**Hallucination（幻覺）**
: 模型生成不基於事實或上下文的虛假資訊。

**Jailbreaking（越獄）**
: 繞過 AI 安全限制的攻擊技術。

**LLM (Large Language Model)**
: 大型語言模型。

## M-P

**PII (Personally Identifiable Information)**
: 個人可識別資訊。

**Precision（精確率）**
: 預測為正類的樣本中，實際為正類的比例。

**Prompt Injection（提示注入）**
: 通過輸入操縱 LLM 行為的攻擊方式。

## Q-R

**RAG (Retrieval-Augmented Generation)**
: 檢索增強生成，結合檢索和生成的技術架構。

**Recall（召回率）**
: 實際為正類的樣本中，被正確預測的比例。

**Reranker（重排序器）**
: 對檢索結果進行二次排序的模型。

**ROUGE**
: 評估文本摘要品質的指標系列。

## S-Z

**Semantic Similarity（語義相似度）**
: 兩段文字在語義層面的相似程度。

**Toxicity（毒性）**
: 輸出中有害、冒犯或不當內容的程度。

**Zero-shot**
: 不提供示例，直接讓模型執行任務。
