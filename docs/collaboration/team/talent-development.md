# 人才培養計畫

建立 AI 評測團隊需要系統性的人才培養計畫。

## 能力模型

```mermaid
graph TB
    subgraph skills["核心能力"]

```

---

## 角色定義

| 角色 | 職責 | 能力要求 |
|------|------|----------|
| 評測工程師 | 設計執行評測 | 程式設計、指標設計 |
| 數據標註員 | 標註審核數據 | 領域知識、細心 |
| 分析師 | 分析評測結果 | 統計分析、可視化 |
| 項目負責人 | 統籌評測項目 | 項目管理、溝通 |

## 成長路徑

```mermaid
graph LR
    A["初級評測員"] --> B["評測工程師"]

    C --> D["評測架構師"]
```

| 級別 | 年資 | 核心能力 |
|------|------|----------|
| 初級 | 0-1年 | 執行評測任務 |
| 中級 | 1-3年 | 設計評測方案 |
| 高級 | 3-5年 | 架構評測體系 |
| 專家 | 5年+ | 引領行業實踐 |
